# INSTRUCTIONS TO USE THE REPOSITORY

### FRONTEND
```
cd Frontend
npm install
npm start
```

### WEBHOOK
```
cd Webhook
```
PASTE THE .env file first
```
python -m venv env
pip install -r requirements.txt
python main.py
```

### BACKEND
```
cd Backend
python -m venv env
pip install -r requirements.txt
python manage.py runserver
```
